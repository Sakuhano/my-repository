TI-84+CE Basics

To turn on your TI-84+CE, press the ON key located at the bottom left corner. When the calculator starts up, you'll see the home screen where you can perform calculations. Use the arrow keys (▲▼◄►) to move around menus and navigate through options.

To turn it off, press 2nd followed by ON (which also functions as the “OFF” command).

[Image: TI-84 Home Screen]

The ALPHA key lets you access the green-colored functions written above many keys. These include letters (useful for naming variables), special commands, and shortcuts.

When you press ALPHA once, the next key you press will use its green function. If you press ALPHA twice quickly, it locks in “Alpha mode” so you can type multiple letters or variables without pressing it repeatedly.

You’ll notice that all the green labels are on the same side of the keyboard as the ALPHA button—this helps you quickly find what’s accessible through it.

[Image: Alpha key example]

The 2ND key gives you access to the blue functions printed above each button. This is one of the most important keys, as it allows you to reach menus and commands that aren’t available on the main key labels.

For instance, pressing 2ND and then MODE opens the “QUIT” command, which exits menus or screens back to the main interface.

[Image: 2ND key usage]

To type letters, press ALPHA and then the corresponding key with the green letter printed above it. You can use letters for variable names (like A, B, X), labeling graphs, or when writing programs.

Remember: if you double-tap ALPHA, it will stay locked in letter mode until you press it again.

[Image: Typing letters on TI-84]

Press the Y= button near the top row to open the function editor, where you can enter equations you’d like to graph. Use the arrow keys to navigate between lines and enter up to ten functions at once.

Once entered, press the GRAPH key to view the plotted functions on the graphing screen. You can also adjust the view by pressing ZOOM or manually change the window with the WINDOW key.

[Image: Graph screen example]

The arrow keys allow you to scroll through menus, move across text boxes, and navigate graphs. In menus, pressing the number next to an option will select it quickly. You can also use ENTER to confirm or select highlighted items.

To exit a menu or go back to the home screen, press 2ND then MODE (QUIT).
