Working with Functions

Graphing, analyzing, and understanding equations on the TI-84+CE

Home Getting Started Capabilities Functions

Opening the Y= Menu

The Y= button opens the function editor, where you can enter equations you want to graph. Each line (Y1, Y2, etc.) represents a separate function. You can toggle any function on or off by highlighting the equals sign and pressing ENTER. This is the main area where graphing begins.

Entering Functions

To enter functions, simply type using the keypad. The calculator uses standard syntax, such as x^2 for x squared or sin(x) for sine. If you need variables, use the X,T,θ,n key. The color and style of each graph can also be changed by selecting the icon before Y=.

Using 2nd and Alpha

Pressing 2nd while in Y= allows access to alternate functions like deleting all entries or inserting a new equation line. The ALPHA key lets you type letters into equations, useful for defining stored variables like A, B, or C.

Adjusting Window

The WINDOW key determines the visible portion of the graph. You can set Xmin, Xmax, Ymin, and Ymax to control your graph’s size. A typical setting like Xmin=-10, Xmax=10, Ymin=-10, Ymax=10 gives a balanced view for most equations.

Using Zoom Options

Press ZOOM to access built-in presets like ZoomStandard, ZoomFit, and ZoomStat. ZoomStandard resets your graph to a 10x10 view, while ZoomFit automatically adjusts the Y-scale to match your equation’s range. These shortcuts save time and keep your graphs centered.

Using TRACE

Press TRACE to move along a graph and view coordinates for specific points. Use the arrow keys to move left or right and check exact X and Y values. This is useful for exploring where your graph increases, decreases, or crosses the axes.

Analyzing with CALC

Press 2nd + TRACE to open the CALC menu. This powerful tool lets you find zeros, minimums, maximums, intersections, and more. You simply select the option, then move the cursor near the point of interest to calculate precise values.

Viewing Tables

Press 2nd + GRAPH to view the table of values for your equations. You can scroll through X and Y pairs or adjust the table settings under 2nd + WINDOW. This helps you compare numeric patterns or test values before graphing.

Editing Table Settings

Within TBLSET (2nd + WINDOW), you can change the starting X value and increments between each step. Setting ΔTbl to smaller values gives you more precision when analyzing function changes numerically.

Graph Styles

You can change how each graph looks by selecting the small icon to the left of Y=. Choose between solid, dotted, or thick lines, and even select colors if your calculator supports them. This helps distinguish multiple graphs at once.

Secondary Functions

Nearly every key has a secondary function accessible by pressing 2nd. For graphing, this allows shortcuts like accessing the CALC menu, adjusting format settings, or tracing more efficiently. Learning these combinations saves time and enhances precision.
